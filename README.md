# 15PuzzleAIProject
Artificial Intelligence Project - 15 Puzzle

A* Search
4 x 4 Board
Tiles: 1 - 15 (w/ 0 being blank tile)
Actions: 
1. Left
2. Up-Left
3. Up
4. Up-Right
5. Right
6. Down-Right
7. Down
8. Down-Left
Heuristic: Sum of Chessboard Distances

n: Initial State
m: Goal State
d: Depth of solution
N: Number of nodes generated
A: Actions taken to reach solution
f: f values of nodes along solution path
